Unsung Hero
The two most problem resolvingiest programs are the Winsock TCP/IP fixer and netsh fix.  Netsh is not really a program that you download as much as a command that you issue by starting a command prompt and typing "netsh int ip reset resetlog.txt"  These two fixes usually take care of any issues leftover after removing spam and viruses.
